# yt-channels-archive


## Installation

```bash
pip install -r requirements-dev.txt
```

## Usage

```bash
yt_archive --output-dir ./backup https://www.youtube.com/channel/UC7edjYPNhTm5LYJMT7UMt0Q/videos https://www.youtube.com/channel/UC6cMYsKMx6XicFcFm7mTsmA
```

## Running the tests

```bash
pytest tests
```

## Running the linter

```bash
flake8
```
